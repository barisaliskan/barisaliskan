### 📌 Featured project

#### [🧪 FaceWell](https://github.com/barisaliskan/FaceWell) &nbsp;·&nbsp; Multi-angle facial wellness analysis

A guided 6-angle webcam scan + a **classical image-processing pipeline** (CLAHE,
histograms, morphology, Canny, color-space math) that produces a personal wellness
dashboard — skin, eyes, stress, symmetry, hydration and dominant mood. MediaPipe FaceMesh
is used *only* for landmark localization; every metric is computed with classical CV.

`Python` · `OpenCV` · `NumPy` · `MediaPipe`
